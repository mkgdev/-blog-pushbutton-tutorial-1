[Tutorial] How to use pushbutton with arduino to on/off LED

What is Arduino?

Arduino is an open source, computer hardware and software company, project, and user community that designs and manufactures microcontroller kits for building digital devices and interactive objects that can sense and control objects in the physical world. Basically you can use your input and output devices with arduino to have fun!!

Introduction

A pushbutton is basically a input device that connect the two points in a circuit when you pressed them. In this tutorial when a pushbutton is pressed the LED will gonna On and Off when you release the pushbutton.

Lets have a look to a internal circuit of pushbutton

As I have earlier mentioned that pushbutton connects two point when pressed.This is how its internal circuit looks like. Two legs on each side are connected together with a opening in between the pairs of connected legs.

How to check which legs are connected in series

You can use the multimeter to identify the series connection in between legs. Set your multimeter to resistance mode and use your test probes in between any two legs. If multimeter gives some positive value that means the legs are connected in series or if it give zero value that means the legs are not connected in series.

What stuffs are required

  • Arduino uno
  • breadboard
  • LED(1)
  • 10k ohm resistor(1)
  • 220 ohm resistor(1)
  • Push button(1)
  • Some Jumper wires

Step-by-Step Instruction

  1. Connect an Arduino GND pin and VCC(+5V) to the long power rails on the breadboard. These will be the GND and VCC rails.
  2. Connect the digital I/O pin 12 to one of the row of breadboard.
  3. Place your LED on breadboard. Connect the shorter(-ve) terminal of LED to GND row and longer(+ve) terminal to Pin 12 row along with 220 Ohm resistor .
  4. Place the push button on breadboard.
  5. Connect one of the terminal(legs connected in series) to GND along with 10k Ohm resistor.
  6. Connect the other terminal to the VCC row of breadboard using jumper wires.
  7. Now connect the grounded terminal of pushbutton to arduino digital I/O pin 2 using jumper wires.
  8. Plug the Arduino board into your computer with a USB cable.
  9. Open the Arduino IDE.
  10. Put the code provided into the code section below.
  11. Compile the code.
  12. Upload the code to Arduino board.
  13. Here you Go! When you press the pushbutton the LED turns ON and when you release the LED turns OFF.

Code

Arduino Pushbutton Code

//Code for pushbutton
int ledpin=12; //For LED
int buttonpin=2; //For Pushbutton
void setup() {
// put your setup code here, to run once:
pinMode(buttonpin,INPUT);
pinMode(ledpin,OUTPUT);
}
void loop() {
// put your main code here, to run repeatedly:
if(digitalRead(buttonpin)==1) //when the pushbutton pressed
{
digitalWrite(ledpin,HIGH);
}
else
{
digitalWrite(ledpin,LOW); //when the pushbutton released
}
}
//end of the code
